# MSE
MSE是一个用C++编写的简单的搜索引擎

##  项目介绍

MSE 主要分为分词、构建索引、和搜索等模块。

解析模块采用流式的 XML 解析工具减少内存的消耗。

索引采用倒排索引的形式,使用了 STL 中 map、list 进行倒排项、倒排列表和倒排索引的构建。

为了数据库可更换,封装的 SQL 的部分操作。

搜索模块采用 Document-at-a-time 的处理方法,实现了在搜索词较少时的高效搜索。
